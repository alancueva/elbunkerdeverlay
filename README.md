# El Bunker de Verlay

Este es un proyecto creado con Next.js e inicializado con `create-next-app`.

## Sobre el Proyecto

Página web para el resturante cevicheria el bunker de verlay

### Tecnologías Utilizadas

*   Next.js
*   React
*   TypeScript

## Primeros Pasos

Sigue estos pasos para tener una copia local del proyecto funcionando.

### Prerrequisitos

Asegúrate de tener instalado Node.js en tu sistema. Puedes descargarlo desde nodejs.org.

### Instalación

1.  Clona el repositorio:
    ```bash
    git clone https://github.com/tu-usuario/elbunkerdeverlay.git
    ```
2.  Navega al directorio del proyecto:
    ```bash
    cd elbunkerdeverlay
    ```
3.  Instala las dependencias:
    ```bash
    npm install
    # o
    yarn install
    # o
    pnpm install
    # o
    bun install
    ```

### Ejecutar el Servidor de Desarrollo

Una vez instaladas las dependencias, puedes iniciar el servidor de desarrollo:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
